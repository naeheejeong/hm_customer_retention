# H&M 고객 유지 전략 분석

> 패션 커머스 · CRM · 고객 세분화

H&M 구매 데이터 기반 고객 세분화(RFM)와 리텐션 전략을 설계한 프로젝트

## 🎯 배경 · 문제정의
- 패션 커머스에서 고객 충성도를 어떻게 정의하고 유지할 것인가
- 구매 데이터의 이상 패턴을 규명하고, 등급별로 차별화된 유지 전략 필요

## 🗂 데이터 · 방법
- H&M 거래 데이터 EDA — 이상 패턴을 ANOVA로 통계 검증
- RFM(Recency·Frequency·Monetary) 기반 고객 **A~F 등급 세분화**
- 등급별 구매 행동 특성 분석 → 충성도 재설계

## 📈 결과 · 임팩트
- ANOVA **F=2,930 · p≈0** — 등급 간 구매 행동 차이 통계적으로 유의
- 등급별 리텐션 전략 제안 (충성 고객 유지 · 이탈 위험군 재활성화)

## 🛠 Tech Stack
Python · Pandas · scikit-learn · Tableau
